# DS

DS is a Typescript library for video editing players.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install foobar.

```bash
pip install ds
```

## Usage

```typescript
import { Core } from "ds"

new Core({
  container: "#container",
  video: {
    width: 1920,
    height: 1080
  },
  data: Data
})
```
